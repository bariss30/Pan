# Web Page Screenshot Automator

Bu proje, birden fazla web sayfasının ekran görüntülerini otomatik olarak almak için Selenium WebDriver ve Python kullanır. Kullanıcı, bir dosyadan alınan URL listesi ile her bir site için ekran görüntüsü alabilir. Chrome ve Firefox tarayıcıları ile çalışabilen bu araç, web sayfalarınızın ekran görüntülerini hızlı bir şekilde almanızı sağlar.

## Gereksinimler

Bu projeyi çalıştırmak için aşağıdaki Python kütüphanelerine ihtiyacınız olacak:

- **selenium**: Web tarayıcılarını otomatikleştirmek için kullanılır.
- **webdriver-manager**: WebDriver sürücülerini otomatik olarak indirir.





## Özellikler

- URL listesi ile toplu ekran görüntüsü alma.
- Farklı tarayıcılar (Chrome, Firefox) desteği.
- Headless (başsız) mod ile tarayıcı arayüzü olmadan çalıştırma.
